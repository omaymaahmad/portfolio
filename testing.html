Friends Night In

Phase One Project

For my first project, I worked with my team mates to make a web application that uses API keys to gather recipe and movie data. User's can insert a singular ingredient and the API will provide them with a random recipe, same with the movie API - a random word can be searched and users are presented with a movie. The application was created through JavaScript, Bulma, HTML & CSS, the site is responsive to desktop, mobile and tablet devices. Click here to view the site and here for the GitHub repository.

Work Day Scheduler

Phase One Homework 5

The week 5 homework was one of my personal favourites, focussing on implementing local storage use I created a web application that allows users to schedule their working day. The time blocks were styled to change colour according to time - grey for the past, pink for present and green for the future. After typing the text if users click the "Save Task" button the information is saved to local storage and remains even after refreshing the page. Click here to view the site and here for the GitHub repository.
